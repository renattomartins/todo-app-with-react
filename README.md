# To Do App with React

A simple proof of concept of a To Do Web Application using React, Node and MongoDB.

## To run:
1. Clone this project
1. Backend:
    1. `$ cd backend/`
    1. `$ npm install`
    1. `$ mongod`
    1. `$ npm run dev`
1. Frontend:
    1. `$ cd ../frontend/`
    1. `$ npm install`
    1. `$ npm run dev`
1. Open browser on: http://localhost:8080/
